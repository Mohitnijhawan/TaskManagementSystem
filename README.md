# 🚀 Task Management System

A full-stack Task Management System built using React with TypeScript on the frontend and ASP.NET Core Minimal API on the backend.

---

## 📌 Features

* 🔐 JWT Authentication
* 👥 Role-Based Authorization (Admin & User)
* 🛠️ Admin has full control over tasks
* 📊 Full CRUD operations for tasks
* ⚠️ Global Exception Handling
* 📌 Task status and priority management

---

## 🏗️ Backend

* ASP.NET Core Minimal API
* Entity Framework Core
* SQL Server
* JWT Authentication
* Role-based access control
* Global exception handling

---

## 🎨 Frontend

* React
* TypeScript
* API integration with backend

---

## 🔁 Core Flow

Login → JWT Token → Role Check → Admin/User Access → Task CRUD Operations

---

## 🔥 Highlights

* Secure role-based system (Admin control enabled)
* Clean API structure with exception handling
* Scalable backend using Minimal API architecture
* Responsive frontend built with React + TypeScript
